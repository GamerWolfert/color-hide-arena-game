# Color Hide Arena

Aparte originele Godot 4-game met 3D camouflagegameplay.

## Starten

```bash
cd ~/Projects/color_hide_arena
godot --editor project.godot
```

Of direct:

```bash
godot --path ~/Projects/color_hide_arena
```

## Controls

- W/A/S/D: bewegen
- Shift: sprinten
- Spatie: springen
- R: wisselen tussen Hider en Seeker
- Linkermuisknop: kleur kopiëren of scannen
- Esc: muis vrijgeven

## Wat werkt

- echte 3D-arena;
- third-person speler;
- kleur kopiëren van muren en objecten;
- seeker/hider-rollen;
- timer;
- eigen arena en eigen code.

Dit is een prototype. Multiplayer, bots, animaties en uitgebreid schilderen per lichaamsdeel moeten nog worden toegevoegd.
